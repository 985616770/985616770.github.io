CSS学习路线(二)选择器

发表于 2018-05-02 更新于 2019-12-23 分类于 CSS

选择器

规则结构:

  • 分两个基本部分: 选择器selector和声明块declaration block 组成

  • 声明块: 由一个或多个声明组成,每一个声明都是属性-值对

选择器分为:

  1. 元素选择器
  2. 类选择器
  3. 后代选择器
  4. 通配选择器
  5. ID 选择器
  6. 属性选择器
  7. 伪类选择器
  8. 子元素选择器
  9. 相邻兄弟选择器

元素选择器:往往是 html 元素,但是在 XML 允许创造新的标记语言.

类选择器:应用样式而不考虑具体涉及的元素.

ID 选择器

  • 和类选择器差不多.区别在于只能在文档运用一次,多次不符合规范,

通配选择器

  • 运用在全局,但是不推荐.易出错
  • 选择全部性能消耗大)

后代选择器

  • 运用在父元素的所有子元素上,

子元素选择器

  • 运用在父元素的第一代子元素上

相邻兄弟选择器

  • 相同父元素下,选择紧跟着另一个元素后的元素

属性选择器

  • 根据元素的属性来选择元素
  • 分为四种:
    • 简单属性选择
    • 具体属性选择
    • 部分属性选择
    • 特定元素选择
