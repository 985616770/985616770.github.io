FCC Truncate a string 解决方法

发表于 2018-06-06 更新于 2019-12-25 分类于 刷题

三行搞定

截断字符串

（用瑞兹来截断对面的退路）

如果字符串的长度比指定的参数num长，则把多余的部分用...来表示。

切记，插入到字符串尾部的三个点号也会计入字符串的长度。

但是，如果指定的参数num小于或等于 3，则添加的三个点号不会计入字符串的长度
