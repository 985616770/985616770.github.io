配置

###　静态资源配置

在http中配置对象 server

  • listen : 监听端口好（注意不要重复）

  • server_name： host名称

  • root：资源文件指向

  • autoindex：（布尔）开启索引　

  • add_header Cache-Control "no-cache,must-revalidate": 禁止缓存

location对象

  • add_header:
    • Access-Control-Allow-Origin *: 允许跨域访问
