事件

定义 : 文档或浏览器窗口中发生的一些特定的交互瞬间

事件分类

写在行内样式中

事件增删

添加事件

  • addEventListener;

  • attachEvent;

移出事件

  • removeEventListener;
  • detachEvent;

区别

  • HTML 事件:强耦合,不利于复用代码

  • DOM 0 级事件:松耦合,代码 js 代码分离

  • 不能绑定多次绑定同一类型事件

  • DOM2 级事件 :优点全有,增加了冒泡捕获

组成

  1. 事件对象

  2. 事件对象绑定事件

  3. 事件句柄:事件处理函数,事件监听函数

冒泡与捕获

捕获:

  • 点击子元素,会先触发父元素的事件,再依次向下触子元素的事件

冒泡:

  • 直系亲属树结构,点底部树结构的某个元素,(亲属树上)的父级元素会依次向上触发

事件委托:

  • 利用冒泡,点击子元素触发父元素的事件委托对象(event)
event
